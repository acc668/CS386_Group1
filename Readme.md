## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/acc668/CS386_Group1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CS386_Group1
   ```
3. Set up dependencies
   ```bash
   cd Q-upSite/
   npm install
   
   cd CPP_CS386_PROJ/
   ```
   ```bash
   For Linux:
   sudo apt update
   sudo apt install g++
   ```
   ```bash
   For Mac:
   xcode-select --install
   ```
   ```bash
   For Windows:
   Install MinGW
   ```
